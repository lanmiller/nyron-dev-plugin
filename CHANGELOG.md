## 0.1.2 — 2026-07-17

- **nyron-waves + qa-handoff: реальная статус-машина DEV** (находка боевой
  обкатки груминга): волна сдаёт тикет в «Готово к тестированию (Ревью)»,
  вердикт «ДОРАБОТКА» → статус «Доработка», «Тестирование» = проверка
  постановщиком после «ПРИНЯТО». Справочно вписаны transition id эпик-доски.
- qa-handoff: статус эпика при передаче назван точным именем воркфлоу.

## 0.1.1 — 2026-07-17

- README: правило версий (любая правка контента = бамп тем же коммитом).

## 0.1.0 — 2026-07-16

- Первый релиз: 4 скилла — `jira-task-standard` (JTBD/MECE/McKinsey/два слоя/
  DoD/«Как тестировать» + декомпозиция бизнес-задач + суперповеры на груминге),
  `nyron-waves` (диспетчер волн через Jira-шину), `live-epic-testing`
  (посекундная методика live-прогонов с картой флоу), `qa-handoff`
  (QA-приёмка эпика одним тикетом).
- Дистрибуция: GitLab (канон) → push mirror → GitHub `lanmiller/nyron-dev-plugin`
  → маркетплейс Claude Desktop/Code.
