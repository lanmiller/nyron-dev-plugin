## 0.1.3 — 2026-07-17

Итоги обкатки ночной пачки волн 17.07 (6 волн, 17/17 тикетов приняты к утру).

- **nyron-waves: активный мониторинг Jira диспетчером** (закрыт ночной пробел —
  волны перевели 17 тикетов «В работе» за 5 минут, диспетчер узнал от
  пользователя): контрольный JQL-чек через 5–10 мин после старта чипов; JQL по
  тикетам волн ПЕРВЫМ действием на каждом пробуждении; опциональный Jira-вотчер
  на REST-токене (дефолт остаётся бес-токенным).
- **nyron-waves: гибрид вотчеров — канон вместо sleep-полла** (латентность
  сдача→ревью ~3 мин вместо 15–25): фоновые until-loop'ы будят сессию по
  событию, ScheduleWakeup — только страховочный пульс. Правила: фильтр эха
  своих комментов, перевзвод только отдельным трекаемым run_in_background,
  волна в ожидании вердикта по А пилит Б.
- **nyron-waves: коммит-конвенция волн** — заголовок коммита начинается с ключа
  тикета (`DEV-XXX: …`), пачка тикетов — все ключи списком; те же ключи в MR
  и отчёт-комменте.
- **Atlassian-доступ канонизирован**: только плагинный MCP (`plugin:atlassian`,
  грант на nyron), проверка гранта первым ходом — в профиле бывает второй
  коннектор без гранта, ночью сессии в него втыкались.
- **nyron-waves: уроки эксплуатации** — зависшие ревью-агенты (перезапуск узким
  скоупом, после 2 фейлов — руками), критерий смерти волны и takeover поверх
  её веток, критичные правила в бриф ДО старта, vitest-грабля worktree фронта,
  бранч-гигиена после мержа пачки; тезис «Jira = событийная шина» вписан якорем.
- **live-epic-testing: грабли мониторов** — буферизация в пайпах
  (`--line-buffered`/`stdbuf`), отправка чата только Enter-keydown, второй
  коннект юзера крадёт HITL.
- README: известная грабля рассинхрона кэша Desktop ↔ CLI; nyron-dev/README —
  актуальная установка (отдельный репо вместо docs-репо) и требования доступов.
- Канон `ai-evolve-docs-test/shared/wave-rules.md` синхронизирован с шаблоном
  (коммит-конвенция, вотчер вместо полла, vitest-грабля, проверка Atlassian).

## 0.1.2 — 2026-07-17

- **nyron-waves + qa-handoff: реальная статус-машина DEV** (находка боевой

## 0.1.0 — 2026-07-16

- Первый релиз: 4 скилла — `jira-task-standard` (JTBD/MECE/McKinsey/два слоя/

  (QA-приёмка эпика одним тикетом).

  → маркетплейс Claude Desktop/Code.
