# Замер по гипотезе {ID} — {Название}

> Стадия «замер» → «вердикт». Меряем **ровно те метрики, что заявлены в блоке 9**
> карточки (D в HADI) — заявленные заранее, не подобранные постфактум. Каждая
> цифра — с пруфом (дашборд, запрос, выгрузка) и датой.

**Дата раскатки:** YYYY-MM-DD · **Источник:** <дашборд/запрос> · **Метод:** дашборд / A/B

## Гейт «есть чем мерить» (до старта замера)

- [ ] события шлются
- [ ] дашборд/отчёт собран
- [ ] разметка на месте (сегменты, версии, флаги)

_Не сходится → гипотеза возвращается в постановку задачей аналитике, а не
замеряется «как получится»._

## Вердикт (Insight в HADI)

**Решение:** масштабировать / итерировать / закрыть.

**Инсайт:** _не «зашло / не зашло», а **что узнали** про пользователя или
механику — переиспользуемо в следующих гипотезах._

**Куда положили:** <файл канона / журнал> — инсайт без записи в канон
не существует.

**Закрыт навсегда?** Да → стадия карточки «не переоткрывать» + строка в журнал.
